# 🏪 Ma Boutique

Un petit site web créé par un étudiant en maths-infos passionné de tout ce qui a un rapport avec l'informatique.

## 📋 Description

**Ma Boutique** est un site e-commerce moderne et responsive conçu pour présenter des produits avec une interface utilisateur épurée et conviviale. Le site est entièrement développé en HTML5, CSS3 et JavaScript vanilla.

## ✨ Fonctionnalités

- ✅ **Design moderne et responsive** - S'adapte à tous les appareils (mobile, tablette, desktop)
- ✅ **Navigation fluide** - Menu sticky qui suit le scroll
- ✅ **Grille de produits** - Affichage élégant avec animations au scroll
- ✅ **Formulaire de contact** - Validation complète des données
- ✅ **Intégration WhatsApp** - Chat direct via WhatsApp Business
- ✅ **Bouton scroll vers le haut** - Pour une meilleure navigation
- ✅ **Animations CSS** - Transitions fluides et professionnelles
- ✅ **SEO optimisé** - Meta tags pour meilleur référencement

## 🛠️ Stack Technologique

- **Frontend :** HTML5, CSS3, JavaScript ES6
- **Design :** Responsive, Mobile-First
- **Animations :** CSS3 + Intersection Observer API
- **Images :** Unsplash API (images de haute qualité)

## 🚀 Comment utiliser

### Option 1 : Ouvrir localement
1. Téléchargez les fichiers
2. Ouvrez `index.html` dans votre navigateur
3. C'est tout ! 🎉

### Option 2 : Sur un serveur local
```bash
# Avec Python 3
python -m http.server 8000

# Puis accédez à http://localhost:8000
```

## 📋 Fonctionnalités détaillées

### 1. Navigation
- Menu sticky qui reste visible lors du scroll
- Liens de navigation vers les sections principales
- Hover effects pour meilleure UX

- **Validation en temps réel** des champs
